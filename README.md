### ◈ 학교폭력 영향인자

가정요인 (맞벌이, 소득수준, 결손가정 여부)

학교요인 (음주, 흡연, 성적)

환경요인 (유해환경, 폭력게임, 음란물 노출)

→ 학교폭력 및 청소년 범죄 증가

2) **관계시각화**를 통한 인자 간 관계 도출
산점도, 상관계수 구하기, 회귀분석
